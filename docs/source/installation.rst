Installation
============


Install from PyPI
^^^^^^^^^^^^^^^^^^
It is recommended to use ``pip`` for installation. You need to check the upgrade at the beginning:

.. code-block:: console

   pip install --upgrade pip

Then you can instll this package using the code:

.. code-block:: console

   pip install GENetLib
